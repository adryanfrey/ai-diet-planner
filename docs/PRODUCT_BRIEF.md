# AI Diet Planner - Product Brief

## Project Overview

AI Diet Planner is a web application that generates personalized daily meal plans for users. The application uses AI to create customized diet plans based on user preferences, dietary restrictions, and health goals. Users can interact with the generated meal plans through an iterative refinement process, allowing them to ask questions and make adjustments to better suit their needs.

## Target Audience

- Individuals seeking personalized nutrition guidance
- People with specific dietary restrictions or preferences
- Health-conscious users looking for structured meal planning
- Users who want AI-powered assistance in creating balanced meal plans

## Primary Benefits / Features

1. **Interactive Questionnaire**: Captures user preferences, dietary restrictions, health goals, and nutritional requirements
2. **Personalized Meal Plan Generation**: Creates 3 daily meal plans tailored to individual user responses
3. **AI-Powered Customization**: Leverages AI to understand user needs and generate appropriate meal suggestions
4. **Iterative Refinement (Coming Soon)**: Users will be able to ask questions and adjust individual meals within their generated plans

## High-Level Tech/Architecture

- **Frontend Framework**: React Router v7 (full-stack framework)
- **UI Library**: React 19, Mantine UI
- **Language**: TypeScript
- **Build Tool**: Vite
- **Backend**: Python + FastAPI
- **Deployment**: Docker containerization support
